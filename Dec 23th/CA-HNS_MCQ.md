# SECTION A: COMPUTER ARCHITECTURE (50 MCQs)

## Introduction to Computer Architecture

1. Computer architecture mainly deals with:  
   A. Program syntax  
   B. Hardware structure and behavior  
   C. Software testing  
   D. Network security  
   **Answer:** B

2. Which of the following best defines computer architecture?  
   A. Study of algorithms  
   B. Design and organization of computer systems  
   C. Programming languages  
   D. Database management  
   **Answer:** B

3. Why is computer architecture important for software developers?  
   A. To design GUIs  
   B. To write optimized and efficient code  
   C. To manage databases  
   D. To secure networks  
   **Answer:** B

4. Which term refers to the speed of the processor?  
   A. Latency  
   B. Throughput  
   C. Clock speed  
   D. Cache  
   **Answer:** C

5. Which is NOT a basic computer architecture term?  
   A. Register  
   B. Cache  
   C. Compiler  
   D. Bus  
   **Answer:** C

6. The CPU is also known as:  
   A. Brain of the computer  
   B. Memory unit  
   C. Storage unit  
   D. Input unit  
   **Answer:** A

7. Which CPU component performs arithmetic operations?  
   A. Control Unit  
   B. Register  
   C. ALU  
   D. Cache  
   **Answer:** C

8. The Control Unit is responsible for:  
   A. Storing data  
   B. Executing instructions  
   C. Directing operations  
   D. Performing calculations  
   **Answer:** C

9. Which register stores the next instruction address?  
   A. MAR  
   B. MDR  
   C. Program Counter  
   D. Accumulator  
   **Answer:** C

10. Instruction execution follows which cycle?  
    A. Decode–Fetch–Execute  
    B. Execute–Fetch–Decode  
    C. Fetch–Decode–Execute  
    D. Load–Run–Stop  
    **Answer:** C
