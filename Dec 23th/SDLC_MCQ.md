## SDLC Basics & Importance

   A. Software Design Life Cycle  
   B. System Development Life Cycle  
   C. Software Development Life Cycle  
   D. System Design Level Cycle  
   **Answer:** C

   A. Reduce hardware cost  
   B. Improve coding speed only  
   C. Manage and control software development  
   D. Eliminate testing  
   **Answer:** C

   A. Writing complex algorithms  
   B. Managing project risks and quality  
   C. Increasing server speed  
   D. Marketing software  
   **Answer:** B

4. Which of the following best defines SDLC?  
   A. A coding technique  
   B. A testing framework

   D. A deployment tool
