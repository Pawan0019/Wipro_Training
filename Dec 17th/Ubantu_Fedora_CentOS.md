# What is Ubuntu?

Ubuntu is a popular, open-source Linux distribution based on Debian, developed and maintained by Canonical Ltd. It is designed to be user-friendly, secure, and stable, making it one of the most widely used Linux operating systems for desktops, servers, and cloud platforms.

The name Ubuntu comes from an African philosophy meaning “humanity to others”.

---

## History of Ubuntu

- First released: October 2004
- Founder: Mark Shuttleworth
- Built on Debian Linux
- Focuses on ease of use and regular updates

---

## Key Features of Ubuntu

### User-Friendly Interface
- Uses GNOME desktop by default
- Clean, modern, and easy for beginners

### Open Source & Free
- Completely free to use
- Large community support

### Secure
- Regular security updates
- Built-in firewall (UFW)
- AppArmor security framework

### Stable & Reliable
- Suitable for personal and enterprise use
- Long-Term Support (LTS) versions available

### Strong Software Support
- Thousands of applications via APT and Snap
- Excellent support for programming tools

---

# What is Fedora?

Fedora is a free, open-source Linux distribution sponsored by Red Hat and developed by the Fedora Project community. It is known for providing the latest stable open-source technologies and acting as an upstream source for Red Hat Enterprise Linux (RHEL).

Fedora focuses on innovation, security, and modern development tools.

---

## History of Fedora

- First released: 2003
- Sponsored by: Red Hat
- Serves as a testing ground for new technologies before they reach RHEL

---

## Key Features of Fedora

### Cutting-Edge Technology
- Latest Linux kernel
- New versions of programming languages and tools
- Early adoption of system innovations

### Strong Security
- SELinux enabled by default
- Regular and fast security updates

### Developer-Friendly
- Excellent support for containers (Docker, Podman)
- Preinstalled tools for DevOps and cloud development

### Clean & Stable
- Focuses on upstream open-source projects
- Minimal vendor customization

---

## Fedora Editions

| Edition | Purpose |
|-------|--------|
| Fedora Workstation | Developers & desktop users |
| Fedora Server | Servers and data centers |
| Fedora IoT | Embedded & IoT devices |
| Fedora CoreOS | Container-based workloads |

# What is CentOS?

CentOS (Community ENTerprise Operating System) is a free, open-source Linux distribution that was historically built from the source code of Red Hat Enterprise Linux (RHEL). It is known for enterprise-level stability, security, and long-term reliability, making it a popular choice for servers and production environments.

---

## History of CentOS

- Initial release: 2004
- Based on: Red Hat Enterprise Linux (RHEL)
- 2014: Red Hat became the primary sponsor
- 2020: Red Hat announced a shift in CentOS’s direction

---

## CentOS Versions Explained

### CentOS Linux (Discontinued)
- A downstream rebuild of RHEL
- Very stable, long support cycles
- CentOS 8 support ended in December 2021

### CentOS Stream (Current)
- A rolling-release distribution
- Sits between Fedora and RHEL
- Acts as a preview of the next RHEL release
- Continuously updated

---

## CentOS Stream Positioning

Fedora  →  CentOS Stream  →  RHEL  
(latest)    (midstream)      (stable)

---

## Key Features of CentOS Stream

### Enterprise Linux Base
- Closely aligned with RHEL
- Suitable for testing enterprise workloads

### Stable but Rolling
- More stable than Fedora
- Less stable than RHEL

### Strong Server Performance
- Ideal for backend services and servers

### Open Source & Free
- No subscription required

---

## Common Use Cases

- Web servers (Apache, Nginx)
- Database servers
- Application servers
- Enterprise Linux testing
- CI/CD pipelines

---

## CentOS vs Alternatives

| Feature | CentOS Stream | RHEL | AlmaLinux / Rocky Linux |
|-------|---------------|------|-------------------------|
| Release Type | Rolling | Stable | Stable |
| Cost | Free | Paid | Free |
| Enterprise Use | Testing / Dev | Production | Production |
| Support | Community | Red Hat | Community + Vendor |

---

## Important Note (Very Exam-Relevant ⚠️)

CentOS Linux is no longer supported. CentOS Stream has replaced it.  
For RHEL-like stability today, Rocky Linux or AlmaLinux are recommended.
